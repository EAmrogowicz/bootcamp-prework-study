GIT

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • GitHub is a cloud-based hosting service that allows us to keep track of our projects inside repositories that are stored remotely. Using GitHub not only allows us to store our code, but also to easily collaborate with others.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: check if our local branch is in sync with the base branch in GitHub
